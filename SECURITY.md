# Security Policy

## Supported version

Security fixes are applied to the latest published release.

## Reporting

Do not include VK tokens, private media, personal paths, or other sensitive information in a public issue.

Use GitHub's private security advisory for this repository when available. For ordinary non-sensitive bugs, create a GitHub issue with reproduction steps, the Media Extractor for VK version, and the Windows version.

## Downloads

Official binaries are published only under `burletov/vk-media-extractor` in GitHub Releases. `SHA256SUMS.txt` is provided for optional integrity checks of `MediaExtractorForVK.exe`.

The application is not currently code-signed. A SmartScreen warning alone does not prove that a file is malicious, but users should never run copies obtained from unrelated websites.
