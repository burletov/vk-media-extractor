# MediaTidy

[![Release](https://img.shields.io/github/v/release/burletov/MediaTidy)](https://github.com/burletov/MediaTidy/releases/latest)
[![CI](https://github.com/burletov/MediaTidy/actions/workflows/ci.yml/badge.svg)](https://github.com/burletov/MediaTidy/actions/workflows/ci.yml)
[![License](https://img.shields.io/github/license/burletov/MediaTidy)](LICENSE)
[![Windows](https://img.shields.io/badge/Windows-10%2F11-0078D4)](https://github.com/burletov/MediaTidy/releases/latest)

**MediaTidy** — бесплатный локальный менеджер фото и видео для Windows. Он помогает найти точные и визуально похожие файлы, сравнить экземпляры, убрать лишнее в карантин, разобрать медиатеку по категориям и возобновляемо импортировать вложения из VK.

[English version](README.en.md)

## Скачать

1. Откройте страницу [последнего релиза](https://github.com/burletov/MediaTidy/releases/latest).
2. Скачайте `MediaTidy.exe` и находящийся рядом `SHA256SUMS.txt`.
3. Запустите `MediaTidy.exe`. Установка не требуется.

Приложение пока не подписано коммерческим сертификатом, поэтому Windows SmartScreen может показать предупреждение для нового файла. Проверяйте, что файл скачан именно из репозитория `burletov/MediaTidy`, и сверяйте SHA-256:

```powershell
Get-FileHash .\MediaTidy.exe -Algorithm SHA256
```

## Возможности

- поиск точных дубликатов по SHA-256;
- поиск визуально похожих изображений;
- поддержка фото и видеофайлов;
- сортировка таблицы по столбцам;
- сравнение двух изображений рядом с синхронным масштабом;
- рекомендация лучшего кадра с учётом резкости и лиц;
- карантин с возможностью восстановления;
- экспорт CSV-отчёта до файловых операций;
- работа сразу с несколькими папками и объединение их содержимого;
- вероятностные категории: фото, скриншот, мем, документ, чек, размытое изображение и графика;
- локальный OCR, EXIF-признаки и переключаемые CLIP-модели;
- импорт фото и видео из всех личных диалогов и бесед VK либо из одного диалога;
- контрольные точки, продолжение импорта и повтор неудачных загрузок;
- светлая и тёмная темы.

MediaTidy **не удаляет файлы автоматически по результатам распознавания**. Категории и оценки модели являются только подсказками.

## Системные требования

- Windows 10 или Windows 11, x64;
- минимум 4 ГБ ОЗУ, рекомендуется 8 ГБ;
- около 400 МБ свободного места для приложения;
- подключение к сети требуется только для импорта VK и загрузки дополнительных моделей.

Обычное сканирование, OCR и распознавание выполняются локально на компьютере.

## Быстрый старт

1. Запустите приложение.
2. Нажмите **«Выбрать папки...»** и добавьте одну или несколько папок.
3. При необходимости откройте **Настройки → Параметры анализа**.
4. Нажмите **«Сканировать»**.
5. Используйте фильтры, сравнение и отчёт перед перемещением файлов.
6. Ненужные экземпляры сначала отправляйте в карантин, откуда их можно восстановить.

## Импорт из VK

1. Откройте **Файл → Импорт из VK**.
2. Укажите собственный токен VK, папку назначения и типы медиа.
3. Выберите импорт всех личных диалогов и бесед либо одного диалога.
4. Для одного чата откройте нужный диалог или беседу VK и скопируйте ID из адресной строки.
5. При необходимости отдельно включите GIF и лимит размера файлов.
6. Запустите импорт. При остановке контрольная точка сохраняется, а следующий запуск продолжит работу.

Получение полного архива может занимать десятки часов. Не удаляйте служебные файлы `.mediatidy-vk-*` и незавершённые `.part`, если планируете продолжить импорт.

MediaTidy является неофициальным приложением и не связано с VK. Кнопка получения токена открывает сторонний сервис VKHost. Проверяйте его адрес и разрешения токена самостоятельно. Не публикуйте токены и не передавайте их другим людям. Пользователь отвечает за соблюдение правил VK и прав на загружаемые материалы.

## Конфиденциальность

- выбранные фото и видео не загружаются разработчику;
- пути, результаты анализа и токен VK не отправляются на сервер MediaTidy;
- токен VK используется для запросов к API VK и не сохраняется приложением;
- дополнительные модели загружаются напрямую с Hugging Face;
- телеметрия и рекламные SDK отсутствуют.

Подробнее: [PRIVACY.md](PRIVACY.md).

## Сборка из исходников

Потребуются Windows 10/11, [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) и PowerShell.

```powershell
git clone https://github.com/burletov/MediaTidy.git
cd MediaTidy
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\download-model.ps1
dotnet restore .\MediaTidy.sln
dotnet run --project .\tests\MediaTidy.SmokeTests\MediaTidy.SmokeTests.csproj -c Release
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\build-release.ps1
```

Готовый автономный EXE и контрольная сумма появятся в `artifacts`.

## Структура проекта

- `src/MediaTidy` — WinForms-приложение;
- `tests/MediaTidy.SmokeTests` — функциональная smoke-проверка;
- `scripts` — загрузка модели и воспроизводимая сборка;
- `docs` — логотип и продуктовый план;
- `.github/workflows` — CI и автоматическая публикация релизов по тегам `v*`.
