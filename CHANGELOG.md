# Media Extractor for VK 2.7.0

## Rebrand

- Проект переименован в **Media Extractor for VK**.
- Репозиторий переименован в `vk-media-extractor`.
- EXE переименован в `MediaExtractorForVK.exe`.
- В README, About, SECURITY, PRIVACY и THIRD_PARTY_NOTICES добавлено явное указание, что приложение неофициальное, не связано с VK и не использует логотип VK.
- Служебные папки и файлы используют имя `MediaExtractorForVK`.

## Что осталось без изменений

- Импорт фото, видео и GIF из VK.
- Возобновляемая загрузка с контрольными точками.
- Поиск точных дубликатов фото и видео по SHA-256.
- Поиск похожих изображений.
- Карантин, восстановление, CSV-отчёт и локальная обработка.

# Media Extractor for VK 2.6.2

## Интерфейс и импорт VK

- Импорт VK использует один последовательный поток загрузки без настройки количества потоков.
- Убраны неинформативные показатели «Последняя активность» и «Осталось».
- Поле ID чата показывается только при выборе импорта одного диалога или беседы.
- Инструкция по ID чата упрощена: нужно скопировать ID из адресной строки VK.
- GIF-файлы вынесены в отдельный чекбокс.
- Добавлен опциональный лимит размера скачиваемых файлов, по умолчанию 1 ГБ.
- Исправлены disabled-состояния в тёмной теме для чекбоксов, переключателей и полей ввода.
- Добавлен быстрый выбор кандидатов из групп похожих изображений с предупреждением перед карантином.
- В выборе папок пояснено, где будет создан карантин при общей медиатеке и при работе с исходными папками.

# Media Extractor for VK 2.6.1

## Интерфейс

- Полностью обновлена тёмная тема: кнопки, стрелки, меню, списки, таблицы и индикаторы прогресса.
- Кнопка «Действия» выровнена с соседними фильтрами по высоте и вертикальной позиции.
- В числовые поля добавлен внутренний отступ слева.
- Добавлены сохранённые профили сканирования.
- Добавлено сравнение двух изображений рядом с синхронным масштабом и перемещением.
- Добавлен экспорт CSV-отчёта до перемещения файлов.

## Импорт VK

- Импорт всех личных сообщений и бесед либо одного диалога по ID/ссылке.
- Автоматические контрольные точки, продолжение после остановки и докачка `.part`-файлов.
- Очередь неудачных загрузок с повтором при следующем запуске.
- Расширенный журнал и предупреждение о длительности импорта.

## Анализ

- Рекомендация лучшего кадра учитывает резкость и количество лиц.
- Добавлены CLIP ViT-B/32, ViT-B/16 и ViT-L/14 с размерами и требованиями к памяти.
- Названия моделей не используют субъективные метки «стандартная» и «усиленная».
